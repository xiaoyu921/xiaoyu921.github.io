1. 基于GIS和空间可达性的城市地价分布研究_宋珂

简述 ：文章数据选取浙江省台州市的主体城区之一路桥区，获取了路桥区的地理底图与建成区的地价抽样数据，以路桥区的建成区为研究区。所有图形数据均以 ArcGIS 的.shapefile 格式储存在 Geodatabase中；地价样点的属性数据以 Access（我没用过） 的.mdb 格式储存在 Geodatabase中，并与地价样点的图形数据一一关联。

1.1 亮点

1.1.1 基于特征点的空间句法
传统的空间句法通过轴线来分割城市系统，分析对象为轴线，沿轴线所有点的空间句法分析变量值是相同的。 这种分析方法不太合逻辑。因为一条街道上各个位置的空间可达性不可能完全相同，所以一条轴线上各点的空间句法变量值也应有所变化。为此，本文提出了一种新的思路——基于特征点的空间句法， 即以一组特征点表示城市结构，以代替轴线。一般而言，地价信息通过样点调查获得，以点状形式存储。这样，将空间句法特征点与地价样点在空间上关联起来，便可分析空间可达性与地价分布之间的关系.
1.1.2 特征点的生成方法

a. 特征点可以借助城市交通网络自动产生

b. 环形道路处理简便

一方面，这些特征点大都可以借助城市交通网络自动产生； 另一方面，基于特征点的空间句法对于环形道路的表示也相对简单一些， 方法是将环形道路分割成为几段弧形，分别取各段弧的顶点即可。特征点生成的步骤可分为三步：第一步，提取城市交通网络的交点和道路的拐点。提取路口、道路拐点的算法较为简单，可在 ArcGIS 中运用“Visual Basic for Application”（ VBA）编写程序自动完成；第二步，生成这些点的连接关系图（图 6），可以假设城市系统是一个简单的二维平面，继而判断任意两点之间是否可视， 即判断任意两点之间的连线是否与地块或建筑物重叠，这一步也可以运用 VBA 编程实现；第三步，对道路拐点进行筛选。之所以要进行第三个步骤，是因为在实际操作中，可能会遇到某条道路存在轻微的、完全可以忽略的曲折，程序仍然会将这些微小弯曲的拐点提取出来，这样会减弱特征点的典型性、增加后续句法变量计算的复杂度。因此，本文通过一个判别函数[3]来判断某个已被提取的拐点是否符合要求（图 7）。

Th < d/l

式中， Th 是自定义的阈值， d 为点 n 到点 n-1 与 n+1 之间连接线的最短距离， l 为点 n-1 与 n+1 之间的距离。若 d/l > Th，则保留点 n；反之，删除点 n，将 n-1 与 n+1 相连，同时更新连接关系图。对所有拐点依次进行判别，直到全部符合要求。

1.2自我总结

1.2.1 文章提出来分析空间可达性与地价分布关系的方法，对我后期地址域的研究还是很有帮助的

  • 特征点————>轴线 表示城市空间结构

    • 交通网络——参数判别法——>自动提取并生成特征点

    • 特征点———Dijlstra 算法(最短路程)——>集成度(每个特征点与其它特征点之间的最短路程)

    • 就近选取特征点——反距离加权插值——>地价样点集成度

    • 150 个样本—— SPSS 软件——>分析空间可达性与地价之间的相关性程度(对所有地价样点的地价变量与整体和局部集成度分别进行皮尔逊相关性分析与多元回归分析，计算它们之间的数理统计关系，求出表示两者关系的回归方程及其相关性指数。)
